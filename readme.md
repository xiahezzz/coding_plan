# 算法入门

* `print_binary` 位运算，循环坐移动。
* `cal_factorial` 计算N阶阶乘之和。
* `select_sort` 选择排序。
* `bubble_sort` 冒泡排序。
* `insert_sort` 插入排序。
* `prefix_sum_arr` 前缀和数组，为了解决多次频繁查找[L,R]区间内的数值和所设计的数值，在数组规模不大时，还可采用建立L-R矩阵的方式来做。
